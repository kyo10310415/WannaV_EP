## 🗄️ データベース設計

### テーブル一覧（6テーブル）

#### 1. users（ユーザー）
| カラム | 型 | 説明 |
|--------|-----|------|
| id | SERIAL | 主キー |
| email | VARCHAR(255) | メールアドレス（ユニーク） |
| password | VARCHAR(255) | ハッシュ化パスワード |
| name | VARCHAR(255) | 氏名 |
| role | VARCHAR(50) | 権限（管理者/クルー/生徒） |
| created_at | TIMESTAMP | 作成日時 |
| last_login | TIMESTAMP | 最終ログイン |

#### 2. courses（コース）
| カラム | 型 | 説明 |
|--------|-----|------|
| id | SERIAL | 主キー |
| title | VARCHAR(255) | コース名 |
| description | TEXT | 説明 |
| order_index | INTEGER | 表示順 |

#### 3. lessons（レッスン）
| カラム | 型 | 説明 |
|--------|-----|------|
| id | SERIAL | 主キー |
| course_id | INTEGER | コースID（外部キー） |
| title | VARCHAR(255) | レッスン名 |
| description | TEXT | 説明 |
| video_filename | VARCHAR(255) | 動画ファイル名 |
| video_url | TEXT | 動画URL |
| order_index | INTEGER | 表示順 |

#### 4. quiz_questions（クイズ問題）
| カラム | 型 | 説明 |
|--------|-----|------|
| id | SERIAL | 主キー |
| lesson_id | INTEGER | レッスンID（外部キー） |
| question | TEXT | 問題文 |
| options | JSONB | 選択肢（配列） |
| correct_answer | INTEGER | 正解番号 |

#### 5. user_progress（進捗）
| カラム | 型 | 説明 |
|--------|-----|------|
| id | SERIAL | 主キー |
| user_id | INTEGER | ユーザーID（外部キー） |
| lesson_id | INTEGER | レッスンID（外部キー） |
| completed | BOOLEAN | 完了フラグ |
| quiz_passed | BOOLEAN | クイズ合格フラグ |
| last_watched_at | TIMESTAMP | 最終視聴日時 |
| quiz_attempts | INTEGER | クイズ挑戦回数 |

#### 6. notifications（通知ログ）
| カラム | 型 | 説明 |
|--------|-----|------|
| id | SERIAL | 主キー |
| user_id | INTEGER | ユーザーID（外部キー） |
| message | TEXT | 通知内容 |
| sent_at | TIMESTAMP | 送信日時 |
